# 🏑 Hockey App

**Hockey App** es una aplicación diseñada para registrar, visualizar y analizar
estadísticas de partidos de hockey en tiempo real.

Está pensada para su uso en cancha por el cuerpo técnico, funciona **offline**
y está optimizada para **teléfonos y tablets Android**.

---

## ✅ Características

- 📊 Tableros de estadísticas por equipo y jugadoras
- 📈 Comparativas de rendimiento
- 🕘 Historial de partidos
- 📲 Diseño preparado para uso en tablets
- ❌ No requiere conexión a internet durante el partido

---

## 📲 Instalación en Android (APK)

1. Descargá la APK desde la sección **Releases** del repositorio
2. Abrí el archivo en tu dispositivo Android
3. Permití la instalación desde esta fuente si Android lo solicita
4. Instalá la app
5. Abrí **Hockey App** desde el ícono

✅ Funciona offline  
✅ Pantalla completa  
✅ No depende de Chrome
